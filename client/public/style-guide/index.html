FarmEvidence — Palette and Chart Style Guide

Rendered live from design-tokens.css and chart-theme.js — this page is the source of truth for how colour and graphs must look everywhere in the app.
The categorical palette below was corrected from the originally-supplied set — 5 of 8 slots read as gray (OKLCH chroma below the categorical floor) and one failed 3:1 contrast on white, verified via the dataviz skill's validator, not eyeballed. Each hue family was held constant and only chroma/lightness nudged until every check passed. "Stone grey" in particular can't pass as a true categorical color — near-zero chroma is definitionally indistinguishable under colorblindness simulation — so it now reads as a muted olive. These are the values live in the app (design-tokens.css, python-service/chart_style.py) today.
